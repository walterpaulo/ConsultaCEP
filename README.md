# ConsultaCEP

Projeto para consulta de código de endereço Posta(CEP) no Brasil.

### Objetivo
Consulta CEP no DB;
Se não tem registos, busca na ViaCEP e salva no banco.

### Tecnologias

* Java 8 +
* Maven 4.0.0
* JSON
* JUnit 4.13.2

### Apoio externo

* ViaCep [https://viacep.com.br/](https://viacep.com.br/)
